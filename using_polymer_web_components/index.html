Using Polymer Web Components

This example demonstrates how to use Polymer web components in a regular web page.

The steps involved to use Polymer are:

  1. Install the Polymer CLI tools
  2. Install the Polymer components you want to use via Bower, NPM, Yarn, etc.
  3. Include the webcomponents-loader.js file to polyfill older browsers
  4. Import the Polymer file into your page
  5. Import any components you want to use, then use the tags in the page
